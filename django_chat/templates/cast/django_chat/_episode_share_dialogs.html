{% comment %} Attribute values inside this textarea must be DOUBLE-escaped: the textarea content is parsed as rcdata, so the browser decodes one layer of character references when reading the textarea's value. Without the second escape, an episode title containing a double quote (or an embed_url containing an ampersand) would round-trip into an invalid attribute value when the snippet is copied. JS mode uses escapeAttr in embed-modal.js for the same. {% endcomment %}
